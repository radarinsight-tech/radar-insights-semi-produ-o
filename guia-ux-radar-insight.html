📖 Glossário Técnico Front-end
Termos que você vai usar ao reportar problemas de layout
Width / Max-width
Largura de um elemento. Max-width limita o tamanho máximo.
Ex: "a coluna Ação está muito estreita"
Padding
Espaço interno entre o conteúdo e a borda do elemento.
Ex: "o texto está colado na borda do card"
Margin
Espaço externo entre elementos.
Ex: "os cards estão muito juntos"
Truncate / Overflow
Texto longo que é cortado com "..." para não quebrar o layout.
Ex: "o nome do protocolo está quebrando a linha"
Flex / Grid
Sistemas de layout para organizar elementos em linha ou grade.
Ex: "quero dois cards lado a lado"
Responsive / Breakpoint
Adaptação do layout para diferentes tamanhos de tela.
Ex: "no celular os botões ficam sobrepostos"
Z-index
Ordem de sobreposição de elementos (qual fica na frente).
Ex: "o popup fica atrás da tabela"
Overflow scroll
Barra de rolagem quando o conteúdo ultrapassa o tamanho do container.
Ex: "quero scroll só na tabela, não na página"
Fixed / Sticky
Elemento que fica fixo na tela mesmo ao rolar a página.
Ex: "o header deve acompanhar o scroll"
Badge / Chip / Tag
Pequeno rótulo colorido para indicar status ou categoria.
Ex: "o status 'Concluído' em verde"
Tooltip
Texto explicativo que aparece ao passar o mouse sobre um elemento.
Ex: "ao passar no ícone, mostrar o significado"
Modal / Popover
Modal bloqueia a tela. Popover abre ao lado do elemento sem bloquear.
Ex: "o histórico abre num popup pequeno"
Skeleton / Loading state
Animação de carregamento enquanto os dados não chegaram.
Ex: "mostrar um placeholder enquanto carrega"
Empty state
Tela/mensagem quando não há dados para exibir.
Ex: "quando não há atendimentos, mostrar mensagem"
Accordion
Seção que expande/recolhe ao clicar no título.
Ex: "o histórico de lotes recolhível"
Sidebar
Painel lateral de navegação.
Ex: "menu lateral da aba Performance"
⚠️ Problemas Comuns e Como Descrever
Reconheça o problema e use o termo certo ao reportar
Coluna apertada / botão cortado
A coluna não tem largura mínima definida. O conteúdo está sendo cortado.
min-width white-space: nowrap
Texto quebrando linha indesejada
O texto não tem instrução para truncar ou ficar em uma linha.
truncate overflow: hidden
Cards muito espaçados / muito juntos
O espaçamento entre elementos precisa de ajuste.
gap margin padding
Página inteira rola, mas queria só a tabela
A tabela precisa de altura fixa com scroll interno.
max-height overflow-y: scroll
Popup aparece atrás de outros elementos
O popup precisa de z-index maior para ficar na frente.
z-index position: fixed
Layout quebra no celular
Os elementos precisam de regras para telas menores.
responsive breakpoint flex-wrap
✅ Checklist — Antes de Reportar um Problema
Use essa lista para garantir que sua solicitação é clara e completa
  • Identifiquei qual tela / aba / componente tem o problema (ex: aba Operação, tabela de atendimentos)
  • Descrevi o que está errado visualmente (ex: coluna Ação muito estreita)
  • Descrevi como deveria ficar (ex: botões lado a lado sem quebrar linha)
  • Mencionei se o problema ocorre sempre ou só em alguns casos (ex: só quando o nome é longo)
  • Indiquei se é desktop, mobile ou ambos
  • Usei pelo menos um termo técnico do glossário para ser mais preciso

🧑‍💼 Mentoria Preventiva — Modo Atendente
Funcionalidade implementada via Lovable • Prompt de referência disponível abaixo
📋 Visão Geral da Funcionalidade
🎯O atendente realiza sua própria mentoria preventiva — analisa seus atendimentos, vê pontos fortes e áreas de melhoria, sem impacto na avaliação oficial
🔐Acesso isolado — o atendente só enxerga seus próprios dados. Outros módulos ficam bloqueados
📊Nota exibida com aviso de "não oficial" — sem campo de bônus ou régua progressiva
📁Histórico pessoal filtrável por mês — até 10 mentorias por mês
🔑 Permissões e Cadastro
Novo role no sistema
🆕ID: mentoria_atendente — adicionado ao Select de roles em Usuários e Permissões
👤Ao cadastrar usuário com este role, selecionar o Atendente vinculado (lista dos atendentes da tabela)
🚫Usuário com este role é redirecionado para /mentoria-preventiva em qualquer tentativa de acessar outra rota
🛡️ Regras de Validação
Validação de PDF
📄Ao importar PDF, o sistema extrai o nome do atendente e compara com o vínculo do usuário logado (case-insensitive)
⚠️Se o nome não corresponder → bloquear arquivo + exibir alerta: "Este atendimento pertence a outro colaborador"
Limite mensal
🔢 Máximo de 10 mentorias por mês — contador sempre visível no header: 0–7 Verde 8–9 Amarelo 10 Bloqueado
🔒Ao atingir 10: upload desabilitado + banner "Limite mensal atingido — renova em 1º do próximo mês"
Segurança (backend)
🔧Edge Function valida attendant_id e limite antes de processar — nunca confia só no front-end
📈 Nota e Histórico
🏷️Nota exibida normalmente por faixas de cor — mas sempre com aviso amarelo: "Esta nota é para sua mentoria pessoal e não tem caráter oficial. Não afeta seu bônus nem avaliação formal."
🚫Campo de bônus (R$, faixa, régua progressiva) não é exibido para este role
📅Seção "Meu Histórico" com tabela: Protocolo · Data · Nota · Pontos Fortes · Oportunidades · Ver detalhes
🔍Filtros do histórico: seletor de mês/ano + busca por protocolo

📝 Templates Prontos para Situações Comuns
Copie e adapte conforme necessário
Coluna estreita na tabela
Na tabela de atendimentos (aba Operação), a coluna AÇÃO está muito estreita quando há dois botões lado a lado. O layout quebra e os botões ficam sobrepostos.

Esperado: a coluna deve ter min-width suficiente para acomodar os botões sem quebrar. Os botões devem ficar sempre em linha (flex, nowrap).

Prioridade: Média.
Scroll interno na tabela
A tabela de atendimentos precisa de scroll interno (overflow-y: scroll) com altura máxima de 400px. A página principal não deve rolar — apenas a tabela internamente.

A scrollbar deve ser fina e sempre visível na lateral direita da tabela.
Card compacto lado a lado
Os cards [Upload] e [Último Lote] devem ficar lado a lado em grid de 2 colunas (proporção 1fr / 1.5fr). Altura máxima de 130px para ambos. Em mobile (< 768px) empilhar verticalmente.
Texto truncado com tooltip
O campo [nome do atendente] deve ser truncado com reticências (truncate / text-overflow: ellipsis) quando ultrapassar a largura da coluna. Ao passar o mouse, exibir o nome completo em tooltip.
